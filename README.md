# Vue3 TodoList Example

## Getting Start
```bash
yarn # install deps

yarn dev # develop
# or
yarn serve # build and running    

yarn test # test watching
yarn unit:coverage # get coverage

yarn e2e # e2e testing, run yarn serve first.
```
